title: 'Transfer Types'
description: 'Supported transfer rails for moving value on/off chain.'
---

Brale supports these transfer types for funding and payouts. Use them in `transfer_type` fields on addresses and transfers.

- **Wire** (domestic)
- **ACH Credit** (standard, same day)
- **ACH Debit** (standard, same day)
- **RTP**

If you need additional rails, contact support@brale.xyz.
