<p class="intro-subheading">
The Brale API supports Brale-issued stablecoins on 15+ popular blockchains alongside native USDC and Paxos-issued stablecoins. Testnets are also supported.

</p>

Base Currency

Base currency is an important concept in the [Transfer](https://brale.mintlify.app/key-concepts/transfers) function of the API. Each stablecoin is bound 1:1 to it's backing base currency. The following Base currencies are supported in the API:

| Currency  | API Identitier | Available for new issuance |
| --------- | -------------- | -------------------------- |
| US Dollar | USD            | ✅                          |
| MXN Peso  | MXN            | ✅                          |

 

If you’re interested in support for additional blockchains or stablecoins, please reach out to [support@brale.xyz](mailto:support@brale.xyz).
