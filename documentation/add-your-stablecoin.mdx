# Add Your Stablecoin

Publish your stablecoin metadata via the Brale Commons repo so it can be indexed on `data.brale.xyz` (with pricing) and made searchable programmatically.

## How to add your token

1. Open the Commons repo: [Brale-xyz/commons](https://github.com/Brale-xyz/commons/tree/main/Data)  
2. Add your token list under `Data/` (follow the existing naming and JSON structure in the folder).  
3. Submit a pull request.  
4. Once accepted, the token is indexed on `https://data.brale.xyz` with price data and discoverable via programmatic search.

## Advantages

- Indexed and priced on `data.brale.xyz` for downstream use.  
- Programmatic discoverability for integrators.  
- Central, versioned source via the Commons repo.  
- Fast updates: ship changes with a PR to `Data/`.
