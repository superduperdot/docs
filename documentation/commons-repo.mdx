# Commons Repo

The Brale Commons repository hosts the Commons Stablecoin Format (CSF) spec and related shared assets.

## Overview

- GitHub: [Brale-xyz/commons](https://github.com/Brale-xyz/commons)
- CSF JSON (raw): [raw.githubusercontent.com/Brale-xyz/commons/.../csf.json](https://raw.githubusercontent.com/Brale-xyz/commons/main/Commons%20Stablecoin%20Format/csf.json)
- Contents: CSF schema, funds-flow examples, data samples, and shared resources for Brale workflows.
- Add your stablecoin: [Submit via Commons](./add-your-stablecoin)

Use this repo to fetch the latest CSF schema and contribute improvements.
